# 컨텍스트 관리 가이드

긴 세션에서 효율적으로 작업하기 위한 가이드입니다.

## 핵심 규칙
- 긴 세션은 작업 품질을 저하시킬 수 있음
- 복잡한 작업은 여러 세션으로 나누기
- HANDOFF.md로 세션 간 인계

## 명령어

### /clear
- Codex 세션 완전 초기화
- HANDOFF.md 없으면 위험!
- 깨끗하게 시작하고 싶을 때

## 권장 패턴

```
작업 시작
    |
3-5개 작업 완료
    |
$handoff (HANDOFF.md 생성)
    |
/clear (초기화)
    |
새 세션에서 HANDOFF.md 읽기
```

## 주의
긴 세션일수록 품질이 저하될 수 있으므로 주기적으로 세션을 정리하세요.
